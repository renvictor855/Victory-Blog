## 前言
建立一个属于自己的博客网站一直是我的理想，从最早使用 WordPress ，到如今使用 Hugo ，博客这个东西对我有着深深的影响。

## WordPress
这是我最早的一次尝试， 2022 年 3 月，在朋友的帮助下，我第一次有了自己的网站，我把它命名为乔治网 （George Website） 。

服务器是当时免费申请的，域名是在 freedom （已经停止运营）免费申请的 ryqi.ml ，当时第一次 Ping 通 ryqi.ml ，我是无比的兴奋， WordPress 发文章非常容易，它不需要复杂的编程语言，服务器的维护我全交给了我的大神朋友，当时我发了几个比较好的教程类文章，但是在不仅之后，这台免费的服务器到底不运转了，我第一次的博客生涯结束。

## Hexo+GitHub+CloudFlare
我的大神朋友他自己有一个静态博客，紧接着， 2022 年 4 月份，他也帮助我建立了一个静态博客，我到现在也十分感谢他。

Hexo + GitHub + CloudFlare 是当时乃至现在最火的个人博客方案，他当时给我采用的也是这套，他帮我安装了 butterfly 的主题，然后给我完善了主题，网站做得十分精美，依旧叫乔治网 （George Website） 。

Hexo 有个缺点，就是一切都需要编程，我当时只会一点点 Python ，主要用是我自己的兴趣，好在 Hexo 发文章只用 Markdown 语言，这也是我第一次接触 Markdown ，我也深深爱上了 Markdown ，以至于后来我选择笔记软件必须支持 Markdown 格式。

紧接着我建立了 RYQi GROUP ，聘请了我大神朋友当我的顾问，我当时写了几篇商业文章，然后通过 Telegram 平台传播，不久我们就有生意了，我们卖了一次节点搭建服务，一个我随手注册的纯数字的 Google Account ，还有帮人解决 Hexo 建站问题，一共在 2 个月不到的时间赚了 120 元钱。（我们每个人 60 元），这也是我第一桶金。

后来由于网站疏于维护+ Freedom 不再提供域名，乔治网 （George Website） 彻底杳无音信……而我们的 RYQi GROUP 也黄了……

## WordPress+Tencent Cloud
2023 年 1 月份，我自己花了 100 余元买了 Tencent Cloud 一年的服务器和 ryqi.xyz 这个域名，一切都没问题，就是卡在了备案这一步，我无奈地放弃了，也亏了 100 多元钱。

## Tencent Cloud的复活
2023 年 2 月份， ChatGPT 爆火，我在一个博主学到了调用送的 API 额度和我服务器来做一个智能微信网友，我用了我一个微信小号和这个服务器做了一个，成功交流过几次，后来疏于维护，也报废了……

## Hello Hexo again!
今年 1 月 3 日，我在观看 YouTube 时，意外地看到了用 Hexo 建站的教程，结合我以前的经历，我不到 1 小时，就建好了并安装了安知鱼主题。我给它取名胜利博客（Victory Blog）

熟悉的 Hexo 让我回想起往昔的无数事情。

我买了 ryqi.top 这个域名，有意味 RYQi GROUP 再创辉煌的含义，但其实吧，当时选完全是因为 ryqi.top 最便宜……

我用 CloudFlare 解析了我的域名，把 ryqi.top 挂在 Victory Blog 上。

但是我很喜欢折腾博客的美化，但是无奈我水平实在太低，好几次的报错让我陷入了崩溃，我最终放弃了（但其实内心并不愿意）。

## Hugo+GitHub+CloudFlare
这就是今天的方案，没啥想说的了，可以去看看更新日志。
